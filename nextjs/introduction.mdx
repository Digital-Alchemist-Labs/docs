# Next.js 소개

Next.js는 React 기반의 풀스택 웹 프레임워크입니다. Vercel에서 개발하고 유지보수하며, 현대적인 웹 애플리케이션을 빠르게 구축할 수 있도록 도와줍니다.

## Next.js란?

Next.js는 React 애플리케이션을 위한 프로덕션 레디 프레임워크입니다. 서버 사이드 렌더링(SSR), 정적 사이트 생성(SSG), 그리고 최신 React 18 기능을 기본적으로 지원합니다.

## 왜 Next.js인가?

### 1. 뛰어난 개발자 경험

  <Card title="Zero Config" icon="gear">

  <Card title="Fast Refresh" icon="rotate">

  </Card>
  <Card title="TypeScript 지원" icon="code">
    TypeScript가 기본 내장되어 있습니다.
  </Card>

### 2. 성능 최적화

- **자동 코드 분할**: 필요한 코드만 로드
- **이미지 최적화**: `next/image`를 통한 자동 최적화
- **폰트 최적화**: 웹 폰트 자동 최적화

### 3. 유연한 렌더링
