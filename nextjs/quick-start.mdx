# 빠른 시작

Next.js App Router를 사용하여 첫 번째 애플리케이션을 만들어봅니다.

## 페이지 만들기

App Router에서 페이지는 `app` 디렉토리 내의 `page.tsx` 파일로 정의됩니다.

### 홈페이지

`src/app/page.tsx`:

```tsx
export default function HomePage() {
  return (
    <main>
      <h1>Welcome to Next.js!</h1>
      <p>This is your first Next.js application.</p>
    </main>
  );
}
```

### 추가 페이지

`src/app/about/page.tsx`:

```tsx
export default function AboutPage() {
  return (
    <main>
      <h1>About Us</h1>
      <p>Learn more about our team.</p>
    </main>
  );
}
```

<Note>이제 `/about` 경로로 접근할 수 있습니다.</Note>

## 레이아웃 사용하기

`src/app/layout.tsx`:

```tsx
export default function RootLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  return (
    <html lang="ko">
      <body>
        <nav>
          <a href="/">Home</a>
          <a href="/about">About</a>
        </nav>
        {children}
      </body>
    </html>
  );
}
```

<Tip>
  레이아웃은 여러 페이지에서 공유되는 UI를 정의하는 데 사용됩니다. 레이아웃은
  상태를 유지하고 리렌더링되지 않습니다.
</Tip>

## 스타일 적용하기

### CSS Modules

<CodeGroup>

```css page.module.css
.title {
  color: #333;
  font-size: 2rem;
}
```

```tsx page.tsx
import styles from "./page.module.css";

export default function HomePage() {
  return <h1 className={styles.title}>Welcome!</h1>;
}
```

</CodeGroup>
